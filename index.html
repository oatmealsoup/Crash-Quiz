Paragraph Crash Test

HTML documents, at least personally, have a maximum paragraph size of 2^26 bytes, exactly, as shown by this function. Hopefully you will agree... Two paragraphs of this size crash the page, but two that are half of the size can load, for some reason I'd like to understand eventually. I adjusted for the paragraph tags by subtracting seven. The function will probably display a maximal ouput if you do not change the inputs. Do not use any input other than the default, unless it is negative, and do not use if you fear a crash. The error code is "invalid string length", so this could be a browser or HTML specific limit...
